## Minimalny pseudokod
```
min(Tablica[], dlugosc){

    min = Tablica[0]
    i = 0
  
    while(i < dlugosc){
      if (Tablica[i] < min){
          min = Tablica[i]
      }
      i++
    }
    return min
}
```
